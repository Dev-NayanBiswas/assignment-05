The DOM (Document Object Model) is an API (Application Programming Interface) that represents and interacts with any HTML or XML-based markup language document. The DOM is a document model loaded in the browser and representing the document as a node tree, or DOM tree, where each node represents part of the document, it can be an element, text string, or comment.The DOM is one of the most-used APIs on the Web because it allows code running in a browser to access and interact with every node in the document. Nodes can be created, moved, and changed. Event listeners can be added to nodes and triggered on the occurrence of a given event.
There are many methods in Javascript to select an element, Each method has it's own way to select an element. like

getElementById()to get an Element by their unique ID

getElementByClassName() to get an element by required class name

getElementsByTagName() it finds the element by their tag name

querySelector() most common used method to find element, it can be either class-Selector or ID-selector

Event delegation is an way in JavaScript that helps us to handle events at the top level in the Nodes, instead of adding event handlers to every child nodes, It gives us access of all children, by attaching a single event listener on a parent element. its the advantage of event bubbling, where events propagate up from the target element to the parent elements in DOM.
